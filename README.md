## 🎮 게임 소개

**CapiRogue**는 기업 경영 시뮬레이션 게임입니다. 당신은 신생 기업의 대표이며, 한정된 자원과 변동하는 시장 상황 속에서 기업을 성장시켜야 합니다.

### 핵심 특징

- **실시간 경제 시뮬레이션**: 부채, 신용등급, 현금흐름 관리
- **전략적 의사결정**: 가격 책정, 인력 관리, 설비 투자 결정
- **다변적 변수**: 경기 순환(호황/안정/불황), 라이벌 기업 경쟁, 도매업체 선택
- **높은 난이도**: 로그라이크의 "한 번의 기회" 원칙 - 파산하면 게임 오버

## 🕹️ 게임 요소

### 기본 구성

| 요소 | 설명 |
|------|------|
| **자금 (Cash)** | 기업의 현금 보유액 |
| **부채 (Debt)** | 상환해야 할 차입금 |
| **신용등급 (Credit Grade)** | A(우수) ~ D(위험) - 이자율 결정 |
| **순이익 (Net Profit)** | 월간 수익/손실 |
| **매출 (Revenue)** | 상품 판매액 |
| **공장 (Factory)** | 생산 설비 및 생산량 관리 |

### 주요 의사결정

#### 1️⃣ **가격 책정 (가격 슬라이더)**
- 손익분기점(BEP) 반영
- 낮은 가격 → 판매량 증가, 낮은 이익
- 높은 가격 → 판매량 감소, 높은 이익

#### 2️⃣ **인력 관리 (HR Hiring)**
- **판매팀**: 판매량 증가
- **마케팅팀**: 시장점유율 확대
- **생산팀**: 공장 효율성 증가
- 한정된 봉급 예산으로 전략적 배치

#### 3️⃣ **공장 관리 (Factory)**
- **건설**: 새로운 공장 건설
- **업그레이드**: 기존 공장 효율성 개선
- **제품 개발**: 신상품 출시

#### 4️⃣ **도매 전략 (Vendor Selection)**
- **저가격 (Cheap)**: 낮은 원가, 낮은 품질
- **표준 (Standard)**: 중간 원가, 중간 품질
- **프리미엄 (Premium)**: 높은 원가, 높은 품질

### 경제 상황 (Economic Cycle)

```
🟢 호황 (Boom)      - 모든 판매 +30%, 부채 이자 낮음
🔵 안정 (Stable)    - 기본 상태
🔴 불황 (Recession) - 판매 -30%, 부채 이자 높음
```

- **AI 어드바이저**: 현재 상황에 대한 조언 메시지
- **경제 배너**: 경기 상황 실시간 표시
- **가격 설정**: 제품 가격 슬라이더/입력창
- **손익분기점 정보**: 현재 BEP 가격 표시
- **인력 관리**: 팀별 고용 버튼
- **공장 제어**: 건설, 업그레이드, 제품 개발
