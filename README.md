## 🎮 게임 소개

**CapiRogue**는 기업 경영 시뮬레이션 게임입니다. 당신은 신생 기업의 대표이며, 한정된 자원과 변동하는 시장 상황 속에서 기업을 성장시켜야 합니다.

### 핵심 특징

- **실시간 경제 시뮬레이션**: 부채, 신용등급, 현금흐름 관리
- **전략적 의사결정**: 가격 책정, 인력 관리, 설비 투자 결정
- **다변적 변수**: 경기 순환(호황/안정/불황), 라이벌 기업 경쟁, 도매업체 선택
- **높은 난이도**: 로그라이크의 "한 번의 기회" 원칙 - 파산하면 게임 오버

## 🏫 경제 교과 성취기준 연계

본 프로젝트는 고등학교 경제 과목 성취기준과 직접 연계되도록 설계되었습니다.

### 1) 자원의 희소성과 합리적 선택
- 관련 성취기준: **[12경제01-03]** 편익과 비용을 고려한 합리적 선택 및 의사결정 능력 계발
- 게임 구현:
	- 제한된 초기 자본 안에서 OEM 하청 업체의 품질/발주 수량을 선택
	- R&D 투자와 단기 마케팅 예산 편성 간 기회비용 비교
	- 선택 결과를 편익·비용 로그로 기록해 의사결정 근거를 학습

선택 모델(확정): **A안(선형 수요 함수)**
- 기본수요 1000, 경기 보정치 0.8~1.2
- 수요 계산: 기본수요에서 가격 탄력 항을 차감하는 선형 모델
- 빠른 턴 템포에서 플레이어가 기회비용을 직관적으로 계산 가능

### 2) 수요와 공급에 의한 시장 가격 결정
- 관련 성취기준: **[12경제02-01]** 수요·공급에 의한 시장 균형 결정과 변동 원리 파악
- 게임 구현:
	- 플레이어 판매가 변화에 따라 소비자 수요량이 실시간 변동
	- 가격 경쟁 결과를 동적 시장점유율 파이차트로 시각화
	- 4가지 기업 유형 라이벌이 등장하며, 악성 재고/적자 발생을 시뮬레이션

라이벌 4유형(확정):
- 저가 대량형: 원가 20% 상시 할인, 품질 업그레이드 제한, 공격성 높음
- 프리미엄형: 불황 수요 타격 절반, 초기 점유율 불리, 공격성 낮음
- 혁신형: R&D 효율 1.5배, 위기 시 파산 리스크 증가, 공격성 중간
- 효율형: 재고 비용 대응 우수, 돌발 변수 대응 약함, 공격성 중간

재고 시스템(확정): **A안(빠른 회전)**
- 보관비: 턴당 원가의 5%
- 악성 재고: 3턴 경과부터
- 강제 폐기: 5턴 경과 시, 원가의 10% 폐기 페널티

### 3) 거시경제 지표와 경기 변동
- 관련 성취기준: **[12경제03-01]**, **[12경제03-03]** 거시 변수 탐색 및 경기 변동 이해
- 게임 구현:
	- 평시·호황기·불황기로 순환하는 경기 사이클 알고리즘
	- 무작위 뉴스 이벤트를 통해 금리·물가 상승 지표 노출
	- 80턴 이후 블랙 스완(경제 위기) 이벤트 발동:
		- 시장 총수요 급감
		- 대출 이자율 급등

경기 사이클(확정): **B안(확률형)**
- 국면 최소 2턴 유지
- 평균 지속: 평시 5턴, 호황 3턴, 불황 4턴
- 국면 종료 시 확률 전이로 다음 국면 결정

블랙 스완(확정): **B안(점진형 트리거 + 점진 회복)**
- 80턴부터 발동 확률이 매턴 누적 증가
- 발동 시 10턴 지속: 시장 수요 50% 급감, 기준금리 +30%p
- 종료 후 3턴 U자형 점진 회복

### 4) 정부 개입과 금융 생활
- 관련 성취기준: **[12경제02-02]** 공공 부문의 경제적 역할과 조세 등 개입 사례 이해
- 게임 구현:
	- 월간 턴 종료 시 자동 법인세 정산
	- 신용등급별 차등 대출 이자율 산출
	- 독점 규제/보조금 지급을 모사한 무작위 정책 이벤트

세금/금리(확정):
- 법인세: **A안(누진세)** 적용
- 신용등급 금리 가산: A +1%p, B +3%p, C +7%p, D +15%p

정책 이벤트(확정): **A안(소규모·고빈도)**
- 매 턴 5% 확률 발동
- 규제/보조금 영향은 순자산의 15% 이내로 캡 적용
- 세부 이벤트 풀 적용:
	- 규제: 독점 과징금, 안전점검, 환경규제
	- 보조금: 고용 보조금, R&D 지원금, 투자세액공제

지표 노출(확정): **A안(대시보드형)**
- 상단 배너에 금리·물가지표·정책 상태 1줄 요약 상시 노출
- 툴팁/리포트에서 상세 로그 제공

추가 반영:
- 라이벌 목록에 기업 유형 배지(저가/프리미엄/혁신/효율) 노출
- 난이도별 수요 탄력성 차등 적용 (Easy 완만, Insane 민감)

## 🕹️ 게임 요소

### 기본 구성

| 요소 | 설명 |
|------|------|
| **자금 (Cash)** | 기업의 현금 보유액 |
| **부채 (Debt)** | 상환해야 할 차입금 |
| **신용등급 (Credit Grade)** | A(우수) ~ D(위험) - 이자율 결정 |
| **순이익 (Net Profit)** | 월간 수익/손실 |
| **매출 (Revenue)** | 상품 판매액 |
| **공장 (Factory)** | 생산 설비 및 생산량 관리 |

### 주요 의사결정

#### 1️⃣ **가격 책정 (가격 슬라이더)**
- 손익분기점(BEP) 반영
- 낮은 가격 → 판매량 증가, 낮은 이익
- 높은 가격 → 판매량 감소, 높은 이익

#### 2️⃣ **인력 관리 (HR Hiring)**
- **판매팀**: 판매량 증가
- **마케팅팀**: 시장점유율 확대
- **생산팀**: 공장 효율성 증가
- 한정된 봉급 예산으로 전략적 배치

#### 3️⃣ **공장 관리 (Factory)**
- **건설**: 새로운 공장 건설
- **업그레이드**: 기존 공장 효율성 개선
- **제품 개발**: 신상품 출시

#### 4️⃣ **도매 전략 (Vendor Selection)**
- **저가격 (Cheap)**: 낮은 원가, 낮은 품질
- **표준 (Standard)**: 중간 원가, 중간 품질
- **프리미엄 (Premium)**: 높은 원가, 높은 품질

### 경제 상황 (Economic Cycle)

```
🟢 호황 (Boom)      - 모든 판매 +30%, 부채 이자 낮음
🔵 안정 (Stable)    - 기본 상태
🔴 불황 (Recession) - 판매 -30%, 부채 이자 높음
```
