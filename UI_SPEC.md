## 3. 각 영역별 UI 명세

### 3.1 상단 타임라인 (`timeline-bar`)

**역할:** 1~120턴 여정의 전체 진행도 시각화

```
[캐피로그] [━━━━━━━━⚔️━━━━━━━━━━━━━━━☠️━━━━━━━━━━━━━━●━━━] [47/120개월]
           1턴                    37턴              80턴   현재
```

| 요소 | 구현 | 설명 |
|:---|:---|:---|
| 바 트랙 | `height: 6px`, `background: --s2` | 전체 120턴 |
| 진행 채움 | `linear-gradient(90deg, --blue, #7cc4ff)` | 현재 턴 비율로 width 갱신 |
| 현재 마커 | `width: 10px`, `border-radius: 50%`, `box-shadow: 0 0 8px rgba(88,166,255,.7)` | 흰 점 + 파란 글로우 |
| 진행 애니메이션 | `transition: width .6s cubic-bezier(.22,1,.36,1)` | 스프링 이징 |
| ⚔️ 37턴 마커 | `left: 30.4%` · `title: "37턴: 성장기 진입 — 경쟁 심화"` | hover 시 tooltip |
| ☠️ 80턴 마커 | `left: 66.1%` · `title: "80턴~: 블랙 스완 경고"` | 블랙스완 활성 시 pulse 애니메이션 |
| 블랙스완 강조 | `.tl-marker-bs-active` → `animation: bsPulse 0.8s` | 0 → 1.5배 scale + 빨간 glow |

### 3.2 좌측 패널 (`panel-left`, 240px)

**역할:** 기업 상태 시각화 + 재무 지표 + 누적 손익 차트

#### 3.2.1 사옥 비주얼 (`hq-block`)

순자산에 따라 4단계로 진화. 진화 시 scale 애니메이션 + 토스트 알림.

| 순자산 | 이모지 | 단계명 | 사옥명 |
|:---:|:---:|:---:|:---:|
| 0 ~ 3천만 미만 | 🏚️ | 창업 초기 | 허름한 차고지 |
| 3천만 ~ 1억 미만 | 🏢 | 초기 성장 | 임대 사무실 |
| 1억 ~ 5억 미만 | 🏬 | 성장기 | IT 빌딩 |
| 5억 이상 | 🏙️ | 대기업 | 도심 마천루 |

**진화 애니메이션:**
```css
/* 진화 감지 시 */
scale(0) rotate(-20deg) → scale(1.3) → scale(1)
transition: 0.45s cubic-bezier(.22,1,.36,1)
```

#### 3.2.2 재무 지표 (`stat-row`)

| 레이블 | ID | 색상 조건 |
|:---|:---|:---|
| 💰 보유 현금 | `lp-capital` | 양수 → `--green` / 음수 → `--red` |
| 🏦 대출금 | `lp-debt` | 항상 `--red` |
| 📈 순자산 | `lp-networth` | 양수/음수 조건부 |
| ⭐ 브랜드 가치 | `lp-brand` | 기본 텍스트 |
| 📊 시장 점유율 | `lp-share` | `--blue` |
| 🛡️ 가격 저항성 | `lp-resist` | 기본 텍스트 |
| 📣 마케팅 인지도 | `lp-awareness` | `--purple` (값 있을 때) |

#### 3.2.3 신용등급 배지

```
[A등급] #00E676    순자산 5억+   연 4.8%   한도 순자산×200%
[B등급] --blue     순자산 1억+   연 6.0%   한도 순자산×100%
[C등급] --yellow   순자산 5천만+ 연 8.4%   한도 순자산×50%
[D등급] --red      그 외         연 14.4%  한도 최대 1천만
```

#### 3.2.4 월별 손익 바 차트

최근 12개월 순이익을 미니 바 차트로 표시.
- 이익 바: `--green` (`bar-pos`)
- 손실 바: `--red` (`bar-neg`)
- 기준선(0): 점선

### 3.3 중앙 패널 (`panel-center`, 1fr)

**역할:** 핵심 조작부 + 경제 상황 + 결재 시스템

#### 3.3.1 AI 비서 바 (`advisor-bar`)

난이도별 캐릭터와 브리핑 말풍선.

| 난이도 | 아바타 | 이름 | 버블 색조 |
|:---:|:---:|:---|:---|
| 이지 | 🤖 | AI 비서 (Easy) | 파란 배경 |
| 노멀 | 📋 | 정보 담당관 | 기본 |
| 하드 | 📡 | 데이터 분석기 | 어두운 |
| 인세인 | 💀 | 노이즈 오라클 | 붉은 |

#### 3.3.2 경제 상황 배너 (`eco-banner`)

현재 경기 국면을 전체 폭 배너로 시각화.

| 국면 | 배경 | 상단 보더 | 아이콘 |
|:---:|:---|:---:|:---:|
| 호황 (boom) | `linear-gradient(#0a2218 → --bg)` | `--green` 2px | 🚀 |
| 평시 (stable) | `linear-gradient(#0c1e2b → --bg)` | `--blue` 2px | ➡️ |
| 불황 (recession) | `linear-gradient(#210a0a → --bg)` | `--red` 2px | 📉 |

블랙스완 활성 시 배지에 `⚠️ 제목… N턴` 표시.

#### 3.3.3 산업 티어 블록 (`tier-block`)

```
┌─────────────────────────────────────────────────┐
│ 산업 티어                         [🔬 R&D]       │
│ 🧺 Tier 1 — 기초 소비재                          │
│ 생필품·필수재. 불황에도 수요 안정.                 │
└─────────────────────────────────────────────────┘
```

- R&D 버튼: 자본 충분 시 `--green` / 부족 시 `--blue`
- 최고 티어(4) 도달 시 버튼 비활성 + opacity 0.5

#### 3.3.4 가격 설정 블록 (`price-block`)

```
판매가 설정                    [도매가(표준형): 20,000원]
[━━━━━━━━━━━━━━━━━━●━━━━━━━━━━━━━━━━] [  45,000 ] 원

[⬇️ 보급형      ] [▶️ 표준형      ] [⬆️ 프리미엄   ]
[ 원가×0.8     ] [ 원가×1.0     ] [ 원가×1.5    ]

📦 발주량  [         100 ] 개
발주 선결제: 2,000,000원 (20,000원/개 × 100개)

[ 발주 원가 ]  [ 개당 마진 ]  [  BEP  ]  [ 마진율 ]
[  2,000,000]  [  +25,000 ]  [  20개  ]  [  55%  ]
```

- 도매가 미만 판매 시: `⛔ 판매가가 도매가보다 낮습니다` 경고 표시

#### 3.3.5 직원 교육 블록 (`hr-block`)

3-트랙 독립 시스템. 각 버튼 행에 pip 표시기(이번 달 사용 여부).

| 버튼 | ID | 비용 | 제한 | 효과 표시 |
|:---|:---:|:---:|:---:|:---|
| 💼 영업/CS 교육 | `hr-sales` | 50만원 | 총 10회·1턴 1회 | 저항성 +2% (N/10회) |
| 📢 마케팅 진행 | `hr-mkt` | 유저 입력 | 1턴 1회 | 예산 입력 + 실시간 미리보기 |
| 🚛 물류/생산 교육 | `hr-prod` | 50만원 | 총 10회·1턴 1회 | 산재위험 -5% (N/10회) |

마케팅 입력창: 예산 입력 시 `브랜드 +Npt · 인지도 +N%` 미리보기 실시간 갱신.

#### 3.3.6 공장 관리 블록 (`factory-block`)

```
[설립 (5억)]  [업그레이드 (1억)]  [품목 변경 (5천)]
✅ 운영 중 · Lv.1 · 산재위험 5%
[🔒 안전관리비 ON (월 500만원) — 클릭하여 OFF]
```

안전비 ON: `--green` 테두리 · OFF: `--yellow` 테두리 + 경고 텍스트

#### 3.3.7 월간 결재 서류 (`doc-section`)

```
📋 월간 결재 서류 — 선택 시 이번 달 마감
┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│ 📄 결재 1    │ │ 📋 결재 2    │ │ 📝 결재 3    │
│ 길가의 쓰레기 │ │ 세무조사 예고 │ │ 사내 연애 발각│
│ 3가지 선택   │ │ 2가지 선택   │ │ 2가지 선택   │
│ 길에서 쓰레기 │ │ 국세청으로부터│ │ 마케팅팀 직원 │
```

- hover: `--blue` 테두리 + 배경 `rgba(88,166,255,.08)` + `translateY(-2px)`
- 선택 후: 선택지 모달 → 결과 팝업(1.5초 자동 닫힘) → 턴 진행
